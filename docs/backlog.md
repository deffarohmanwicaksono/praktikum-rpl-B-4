# Product Backlog

| ID Story | Deskripsi Singkat | Deskripsi | Role | Prioritas (MoSCoW) |
|----------|------------------|-----------|------|--------------------|
| US-01 | Registrasi via SSO UNS | As a user, I want mendaftar dan login SeMart menggunakan akun SSO UNS, so that hanya mahasiswa yang valid yang dapat menggunakan fitur marketplace | User | Must Have |
| US-02 | Upload barang | As a seller, I want mengunggah barang bekas, so that barang saya dapat ditemukan oleh buyer di platform | User (Seller) | Must Have |
| US-03 | Verifikasi barang | As an admin, I want meninjau dan memverifikasi barang yang diunggah seller, so that barang ilegal atau berbahaya tidak beredar di platform | Admin | Must Have |
| US-04 | Pencarian barang | As a buyer, I want mencari barang berdasarkan kata kunci, so that saya dapat menemukan barang yang saya butuhkan dengan cepat | User (Buyer) | Must Have |
| US-05 | Chat dengan seller | As a buyer, I want menghubungi seller melalui fitur chat sebelum melakukan pembelian, so that saya dapat bertanya dan bernegosiasi harga | User (Buyer) | Must Have |
| US-06 | Seller kirim link pembelian | As a seller, I want mengirimkan link pembelian melalui chat, so that buyer dapat melanjutkan ke proses checkout | User (Seller) | Must Have |
| US-07 | Checkout & pembayaran | As a buyer, I want melakukan checkout melalui link pembelian, so that saya dapat menyelesaikan transaksi | User (Buyer) | Must Have |
| US-08 | Tutup penjualan (sold out) | As a seller, I want mengubah status barang menjadi "Sold Out", so that barang yang sudah terjual tidak bisa dibeli lagi | User (Seller) | Must Have |
| US-09 | Detail barang | As a buyer, I want melihat detail lengkap barang, so that saya bisa memastikan kondisi dan spesifikasi | User (Buyer) | Should Have |
| US-10 | Edit & hapus barang | As a seller, I want mengedit atau menghapus barang, so that informasi tetap akurat | User (Seller) | Should Have |
| US-11 | Monitoring admin | As an admin, I want memantau aktivitas marketplace, so that saya dapat melihat performa sistem | Admin | Should Have |
| US-12 | Filter & sorting | As a user, I want memfilter dan mengurutkan barang, so that pencarian lebih cepat | User (Buyer) | Could Have |
| US-13 | Wishlist | As a user, I want menyimpan barang ke wishlist, so that mudah ditemukan kembali | User | Could Have |
| US-14 | Rating & ulasan | As a buyer, I want memberikan rating dan ulasan, so that user lain bisa menilai seller | User (Buyer) | Could Have |
| US-15 | Laporan barang | As a user, I want melaporkan barang mencurigakan, so that marketplace tetap aman | User | Could Have |
| US-16 | Tindak laporan | As an admin, I want menindaklanjuti laporan, so that keamanan marketplace terjaga | Admin | Could Have |
